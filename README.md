# Bandline — IELTS tayyorlash platformasi

Bepul, AI asosidagi IELTS tayyorlash platformasi uchun landing page va loyiha rejasi.

## Hozirgi holat

- `index.html` — asosiy landing page (tayyor, ochish uchun brauzerda fayl oching)

## Brend

- **Nomi:** Bandline
- **Konsepsiya:** "Band shkalasi" — foydalanuvchining hozirgi va maqsad ball (band score) ko'rsatkichini vizual tarzda ko'rsatuvchi signature element
- **Ranglar:**
  - Paper (fon): `#EDEFE8`
  - Ink (matn): `#1F2E2A`
  - Gold (maqsad): `#D9A23B`
  - Sage (hozirgi holat): `#4F7C6B`
- **Shriftlar:** Fraunces (sarlavhalar), Inter (matn), IBM Plex Mono (raqamlar/ma'lumot)

## Asosiy funksiyalar (rejalashtirilgan)

1. **Writing baholovchi** — Task 1/2 insho tahlili (TR, CC, LR, GRA mezonlari bo'yicha)
2. **Speaking simulyatori** — ovozli javob + AI talaffuz/ravonlik tahlili
3. **Mock testlar** — Listening va Reading, cheksiz urinish
4. **Shaxsiy reja** — kunlik vazifalar, maqsad balga qarab moslashadi
5. **Lug'at to'plami** — xatolardan avtomatik flashcard
6. **Progress statistikasi** — bo'limlar bo'yicha o'sish grafigi

## Texnik stack (taklif)

- **Frontend:** React + Tailwind (yoki hozirgi vanilla HTML/CSS asosida kengaytirish)
- **AI:** Claude API — Writing/Speaking baholash uchun
- **Backend:** foydalanuvchi profili, progress va lug'at saqlash uchun database (masalan Supabase/Firebase)

## Keyingi bosqichlar

1. AI Writing baholovchi sahifasi (Claude API integratsiyasi)
2. Speaking simulyatori (ovoz yozish + tahlil)
3. Dashboard (statistika va shaxsiy reja)
4. Mock test moduli
5. Lug'at/flashcard tizimi

## Eslatma

"IELTS" — British Council, IDP va Cambridge Assessment English ro'yxatdan o'tgan savdo belgisi. Bu loyiha ularning rasmiy mahsuloti emas va ular bilan bog'liq emas — shuning uchun brendlash va matnlarda bu farq aniq ko'rsatilishi kerak.
